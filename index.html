Foundations

Typography

Two complementary sans serifs. Inter for body and UI; Open Sauce One for headings and the logotype — a distinct display voice without sacrificing UI readability. Product UI and components use semantic tokens; primitives below are the scale they resolve to.

Foundations

Shadows

Shadows communicate elevation. More shadow = higher up = more important. Use sparingly; most surfaces need only --shadow-sm or --shadow-md.

--shadow-noneflat
--shadow-xshairline lift
--shadow-smcards
--shadow-mddropdowns, tooltips
--shadow-lgpopovers
--shadow-xlmodals
--shadow-innerpressed state

Opacity is encoded as 8-digit hex (#rrggbbaa) so shadow tokens live in the same format as color primitives. The focus ring is its own token — --shadow-focus-ring — derived from --color-action-primary via color-mix().

Foundations

Motion

Motion tokens prevent the "every component has its own timing" problem. Every transition uses one of these, and the whole system respects prefers-reduced-motion.

Duration

TokenValueUsage
--duration-instant0msSkip animation
--duration-fast100msHover states, small feedback
--duration-normal200msButtons, dropdowns, tooltips
--duration-slow300msModals, drawers, panels
--duration-slower500msPage transitions

Easing

TokenCurveUsage
--ease-linearlinearProgress bars, scrubbing
--ease-outcubic-bezier(0, 0, 0.2, 1)Elements entering screen
--ease-incubic-bezier(0.4, 0, 1, 1)Elements leaving screen
--ease-in-outcubic-bezier(0.4, 0, 0.2, 1)State changes, toggles
--ease-springcubic-bezier(0.34, 1.56, 0.64, 1)Playful pop (modals, badges)
--ease-bouncecubic-bezier(0.175, 0.885, 0.32, 1.275)Attention-drawing entrance

A global rule in app.css collapses all transitions to 0.01ms when the user has reduced-motion preferences set — no per-component opt-in required.

Components

Button

Action controls in four variants — primary (brand emphasis), secondary (neutral), destructive (danger), and an icon-only modifier. All share a 32px height, 6px radius, Inter Medium 14/20 label, and consistent state styling.

Primary

Skeuomorphic brand button with gloss, glass rim, and outer ring. Hover brightens the gloss only; active presses to brand-600. 

<button type="button" class="btn btn--primary">Compose</button>
